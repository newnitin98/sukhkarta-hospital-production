# Project Brief — Sukhkarta Hospital Website

## Client
Sukhkarta Hospital Orthopaedic and Cardiac Care Centre, Saoner, Nagpur

## Domain
sukhkartahospital.com

## Objective
Build a production-grade, premium healthcare website for an orthopaedic and cardiac care centre in Saoner, Nagpur. The site must serve as the hospital's primary digital presence, build trust with local patients, and drive appointment enquiries.

## Doctors
- **Dr. Gunjan R. Dhundele** — Orthopaedic Surgeon, Joint Replacement Specialist (15+ yrs, DNB, Gold Medal, FJRS)
- **Dr. Tanu Dhundele** — DM Cardiologist, Interventional Cardiologist (Double Gold Medal)

## Target Audience
- Patients in Saoner, Nagpur district seeking orthopaedic and cardiac care
- Patients needing joint replacement without travelling to Nagpur city
- Patients requiring TMT, ECG, and cardiac screening
- Families managing chronic conditions (arthritis, CAD, hypertension)

## Success Metrics
- Appointment enquiry form submissions
- WhatsApp clicks
- Call link clicks
- Page rankings for local SEO keywords
- Website speed (Core Web Vitals)

## Tech Stack
- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion (animations — Phase 1 minimal)
- React Hook Form + Zod
- Vercel deployment

## Phase
Phase 1: Static frontend, WhatsApp-based appointment enquiry
Phase 2: Backend appointment system (see `graphify/appointment-module-roadmap.md`)
